## vps

翻墙首先得租一个 vps，推荐 [Vultr](https://www.vultr.com/?ref=6960892)，最低价格 \$3.5/月（\$2.5 的只有 ipv6 网络），按时付费，可随时部署和删除，非常划算。可以使用信用卡、微信、支付宝付款。比较快的 location 有日本、新加坡和美国西海岸，可以使用家里的宽带对这些地区[测速](https://www.vultr.com/locations/)。
